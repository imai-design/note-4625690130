Code編に入る前の「git」「VSCode」「Claude Code」の入れ方を、画面ごとに並べました。上から順に、画面のとおりに進めるだけでOKです。打つ文字は枠をそのままコピーできます。

PART 1

git を入れる

git は「作業を記録・保存しておく仕組み」です。これが入っていないと次に進めないので、最初に入れます。

1

ターミナルを開く

キーボードで ⌘（command）＋ スペース を押し、出てきた検索に ターミナル と打って Enter。黒い画面（ターミナル）が開きます。

💡 ターミナル＝パソコンに文字で命令する画面です。怖くありません。
2

この1行を貼り付けて Enter

ターミナル
$ xcode-select --install

コピーして、ターミナルに貼り付け（⌘＋V）て Enter を押します。

3

出てきた画面で「インストール」をクリック

🛠
"xcode-select" コマンドを実行するには、コマンドラインデベロッパツールが必要です。今すぐインストールしますか？
あとでインストール

このような確認が出たら 「インストール」 を押します。利用規約が出たら「同意」。あとは自動でダウンロードが進みます（数分かかります）。

4

入ったか確認する

ターミナル
$ git --version git version 2.39.5

git version … と数字が出れば成功です。

✓ git が入りました。PART 2 へ進みます。
1

ダウンロードページを開く

ブラウザ（Edge や Chrome）で次のページを開きます。

https://git-scm.com/download/win
Git for Windows
⬇ 64-bit Git for Windows Setup

「64-bit Git for Windows Setup」 をクリックして、インストーラを保存します。

2

ダウンロードしたファイルを実行

保存された Git-…-64-bit.exe をダブルクリックして開きます。「このアプリがデバイスに変更を…」と出たら 「はい」。

3

基本そのまま「Next」を押していく

📦
Git Setup設定はすべて初めのまま（おすすめ設定）でOKです。迷わず進めて大丈夫。
BackNext ▶

設定はいじらず 「Next」 を押し続け、最後に 「Install」。終わったら 「Finish」。

4

入ったか確認する

スタートメニューで PowerShell を開き、次の1行を貼り付けて Enter。

PowerShell
PS> git --version git version 2.45.1.windows.1

git version … と数字が出れば成功です。

✓ git が入りました。PART 2 へ進みます。
🧩
PART 2

VSCode を入れる

VSCode は「コードを書いたり見たりするための画面の道具」です。Claude Code を動かす土台になります。Mac も Windows も入れ方はほぼ同じです。

1

ダウンロードページを開く

https://code.visualstudio.com/
Visual Studio Code
⬇ Download for macOS ⬇ Download for Windows

青い 「Download for macOS」「Download for Windows」 ボタンをクリックします（自分のパソコン用が自動で選ばれます）。

2

解凍して、アプリケーションに入れる

ダウンロードした VSCode-darwin… .zip をダブルクリックすると、「Visual Studio Code」 が出てきます。それを 「アプリケーション」 フォルダへドラッグして入れます。

💡 これで Launchpad や Spotlight から VSCode を開けるようになります。
2

インストーラを実行する

ダウンロードした VSCodeUserSetup… .exe をダブルクリック。「同意する」を選び、あとは 「次へ」→「インストール」 と進めます。

💡 「デスクトップにアイコンを作る」にチェックを入れておくと、あとで開きやすいです。
3

VSCode を起動する

アプリケーションからデスクトップかスタートメニューから Visual Studio Code を開きます。青いアイコンの画面が出ればOK。

✓ VSCode が入りました。PART 3 へ進みます。
🤖
PART 3

Claude Code を入れる

いちばん大事なところです。むずかしい操作はありません。Claude に「入れて」と頼むだけで、Claude が自分でインストールを進めてくれます。

1

アプリ版の Claude を開く

パソコンに入れた Claude のアプリ を開き、画面を Code に切り替えます（午前に入れたアプリです）。

2

「Claude Code を入れて」と頼む

入力らんに、次の言葉をそのまま打って送ります。

Claude（Code）
Claude Code を入れて

これが ベース（本体）の入れ方 です。あとは Claude が必要なものを自分で用意してくれます。

3

確認が出たら「承認」で進める

🔐
この操作を許可しますか？Claude がインストールを進めようとしています。
拒否承認

途中で確認が出たら 「承認」 を押して進めます。基本は承認でOKです。

4

使えるか確認する

ターミナル（PowerShell）で次の1行を打って、起動すればOKです。

ターミナルPowerShell
$ PS> claude Welcome to Claude Code
✓ これで全部の準備が完了です。お疲れさまでした。
🎬 もっと詳しく見たいとき：実際の画面で見たい場合は Code Vol.1 の動画 をどうぞ。このガイドと同じ流れを、画面そのままで確認できます。
⚠ つまずいたら：①メモに書く → ②自分の Claude に「○○でエラーが出た。どうすればいい？」と聞く（英語の画面は写真に撮って見せる）→ ③それでも解けなければ担当者に手を挙げる。

準備、おつかれさまでした 🎉

git・VSCode・Claude Code がそろえば、あとは作るだけ。Code編をたのしんでください。
