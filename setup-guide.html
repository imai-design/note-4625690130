PART 0

Claude を使えるようにする（Cowork準備）

午前のいちばん最初の準備です。アカウント作成からアプリの導入まで、この順で全部そろえます。Mac も Windows もほぼ同じです。

1

Claude のアカウントを作る

https://claude.ai
Claude にサインアップ
claude.ai を開く

claude.ai を開いて、メールアドレスでアカウントを作ります。

2

Pro プランに加入する

https://claude.ai/upgrade
Claude Pro
Pro にアップグレード

claude.ai/upgrade から Pro プランに加入します（Cowork・Code を使うために必要です）。

3

Claude アプリをダウンロードして入れる

https://claude.ai/download
Claude Desktop
⬇ Download for Mac ⬇ Download for Windows

claude.ai/download からアプリを入れます。Mac は、出てきた Claude を「アプリケーション」へドラッグ。Windows は、ダウンロードした Claude…Setup.exe をダブルクリックして進めます。

4

Cowork の機能をダウンロードする

Claude のアプリを開いてサインインし、画面の 「Cowork」 ボタンを押します。すると Cowork の機能のダウンロードが始まります。

5

パソコンを再起動する

機能のダウンロードが終わったら、いったんパソコンを再起動します。これで Cowork が使えるようになります。

✓ Claude（Cowork）の準備ができました。PART 1 へ進みます。
🎬 詳しい画面は Cowork Vol.1 の動画でも確認できます。
📦
PART 1

git を入れる

git は「作業を記録・保存しておく仕組み」です。これが入っていないと次に進めないので、最初に入れます。

📥 公式のダウンロードページ：git-scm.com/downloads（Mac版＝こちら ／ Windows版＝こちら）。下の手順でうまくいかないときは、ここから直接入れられます。
1

ターミナルを開く

キーボードで ⌘（command）＋ スペース を押し、出てきた検索に ターミナル と打って Enter。黒い画面（ターミナル）が開きます。

💡 ターミナル＝パソコンに文字で命令する画面です。怖くありません。
2

この1行を貼り付けて Enter

ターミナル
$ xcode-select --install

コピーして、ターミナルに貼り付け（⌘＋V）て Enter を押します。

3

出てきた画面で「インストール」をクリック

🛠
"xcode-select" コマンドを実行するには、コマンドラインデベロッパツールが必要です。今すぐインストールしますか？
あとでインストール

このような確認が出たら 「インストール」 を押します。利用規約が出たら「同意」。あとは自動でダウンロードが進みます（数分かかります）。

4

入ったか確認する

ターミナル
$ git --version git version 2.39.5

git version … と数字が出れば成功です。

✓ git が入りました。PART 2 へ進みます。
1

ダウンロードページを開く

ブラウザ（Edge や Chrome）で次のページを開きます。

https://git-scm.com/download/win
Git for Windows
⬇ 64-bit Git for Windows Setup

「64-bit Git for Windows Setup」 をクリックして、インストーラを保存します。

2

ダウンロードしたファイルを実行

保存された Git-…-64-bit.exe をダブルクリックして開きます。「このアプリがデバイスに変更を…」と出たら 「はい」。

3

基本そのまま「Next」を押していく

📦
Git Setup設定はすべて初めのまま（おすすめ設定）でOKです。迷わず進めて大丈夫。
BackNext ▶

設定はいじらず 「Next」 を押し続け、最後に 「Install」。終わったら 「Finish」。

4

入ったか確認する

スタートメニューで PowerShell を開き、次の1行を貼り付けて Enter。

PowerShell
PS> git --version git version 2.45.1.windows.1

git version … と数字が出れば成功です。

✓ git が入りました。PART 2 へ進みます。
🧩
PART 2

VSCode を入れる

VSCode は「コードを書く・見るための画面の道具」。Claude Code を動かす土台です。いちばん簡単なのは、Claude に頼むことです。

1

Claude アプリの Claude Code に頼む

午前に入れた Claude アプリの Claude Code（画面を Code に切り替え）で、次の言葉をそのまま打って送ります。

Claude（Code）
VS Code をダウンロードして入れて
2

確認が出たら「承認」で進める

Claude がダウンロードとインストールを進めてくれます。途中で確認が出たら 「承認」 を押します。

3

VSCode が開けたらOK

インストールが終わって VSCode が開けば完了です。

✓ VSCode が入りました。PART 3 へ進みます。
💡 自分で入れたいときは、公式ページ code.visualstudio.com からもダウンロードできます。
🤖
PART 3

VS Code で Claude Code を使えるようにする

最後の仕上げです。VS Code に Claude の拡張機能を入れて、VS Code の画面から Claude Code を使えるようにします。

1

VS Code の拡張機能で「Claude」を入れる

VS Code の左側の 拡張機能（四角が4つのアイコン）を開き、検索らんに Claude と入力。出てきた「Claude」を インストール します。

VS Code ＞ 拡張機能（Extensions）
🔍 Claude
Claude
Anthropic 公式の拡張機能Install
2

アカウントを連携する

入れた拡張機能から 「サインイン」 を押し、ふだん使っている Claude のアカウントで連携 します。ブラウザで確認が出たら「許可」して、VS Code に戻ります。

🔗
Claude にサインインしますか？VS Code と Claude アカウントを連携します。
あとでサインイン
3

VS Code から Claude Code が使えるようになる

連携できると、VS Code の画面の中で Claude Code が使えるようになります。これで全部の準備は完了です。

✓ これで全部の準備が完了です。お疲れさまでした。
🎬 もっと詳しく見たいとき：実際の画面で見たい場合は Code Vol.1 の動画 をどうぞ。このガイドと同じ流れを、画面そのままで確認できます。
⚠ つまずいたら：①メモに書く → ②自分の Claude に「○○でエラーが出た。どうすればいい？」と聞く（英語の画面は写真に撮って見せる）→ ③それでも解けなければ担当者に手を挙げる。
